# GameEXpedia

This interactive application lets your post game reviews for PS5.

## Installation

Use [git clone](https://github.com/maion308/game-ex.git) to download this repo

```zsh
git clone https://github.com/maion308/game-ex.git
cd game-ex
npm i
```
